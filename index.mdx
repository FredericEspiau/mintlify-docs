## Overview

CookingAuto is a TypeScript CLI that orchestrates a multi-stage pipeline to convert videos into FSRS-based flashcards:

1. **Download** videos from Google Photos (or use local files)
2. **Convert** video to audio via FFmpeg
3. **Transcribe** with speaker diarization (WhisperX or Pyannote)
4. **Generate** flashcards via Gemini AI
5. **Store** in FSRS-based spaced repetition database

## Quick Commands

```bash
# Run main pipeline
pnpm start -m cooking -d 2024-01-15

# Interactive review session
pnpm start review -l 20

# List due cards
pnpm start list --due

# View statistics
pnpm start stats
```
