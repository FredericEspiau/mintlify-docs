## Prerequisites

Before starting, ensure you have:

<AccordionGroup>
  <Accordion icon="node-js" title="Node.js >= 22.21.0">
    ```bash
    node --version  # Should be >= 22.21.0
    ```
    
    Install via [nvm](https://github.com/nvm-sh/nvm) or [fnm](https://github.com/Schniz/fnm).
  </Accordion>
  
  <Accordion icon="waveform-lines" title="FFmpeg">
    Required for video-to-audio conversion.
    
    ```bash
    # macOS
    brew install ffmpeg
    
    # Ubuntu/Debian
    sudo apt install ffmpeg
    
    # Verify
    ffmpeg -version
    ```
  </Accordion>
  
  <Accordion icon="microphone" title="Transcription Engine (choose one)">
    **WhisperX (local):**
    ```bash
    pip install whisperx
    ```
    Requires `HUGGING_FACE_TOKEN` for speaker diarization.
    
    **Pyannote (cloud):**
    Requires `PYANNOTE_KEY` environment variable.
  </Accordion>
  
  <Accordion icon="google" title="Google Cloud Credentials">
    For Google Photos integration:
    
    1. Create OAuth2 Desktop credentials in [Google Cloud Console](https://console.cloud.google.com/)
    2. Enable Google Photos Picker API
    3. Download credentials to `src/auth/credentials.json`
  </Accordion>
</AccordionGroup>

## Installation

```bash
# Clone repository
git clone https://github.com/your-org/cookingauto.git
cd cookingauto

# Install dependencies
pnpm install

# Set up environment
cp .env.example .env
```

## Environment Variables

<Note>
At minimum, you need `GEMINI_API_KEY` and one transcription token.
</Note>

```bash
# Required
GEMINI_API_KEY=your-api-key          # From https://aistudio.google.com/apikey

# Transcription (one required)
HUGGING_FACE_TOKEN=your-token        # For WhisperX diarization
PYANNOTE_KEY=your-key                # For Pyannote cloud API

# Optional
GEMINI_MODEL=gemini-2.5-flash        # Default model
DEBUG=true                           # Enable debug logging
```

## First Pipeline Run

### Option 1: From Google Photos

```bash
# Download video from a specific date and process
pnpm start -m cooking -d 2024-01-15
```

This will:
1. Open a browser for Google OAuth authentication
2. Launch the Google Photos Picker for video selection
3. Download the selected video
4. Convert to audio, transcribe, and generate flashcards

### Option 2: From Local Video

```bash
pnpm start -m cooking --video-file ./my-video.mp4
```

### Option 3: From Existing Audio

```bash
pnpm start -m cooking --audio-file ./my-audio.mp3
```

### Option 4: From Existing Transcript

```bash
pnpm start -m cooking --transcript-file ./transcript.md
```

## Review Your Cards

After generating cards, start an interactive review session:

```bash
pnpm start review -l 10  # Review up to 10 cards
```

Use the number keys to rate each card:
- `1` - Again (forgot completely)
- `2` - Hard (difficult to recall)
- `3` - Good (recalled with effort)
- `4` - Easy (recalled instantly)
